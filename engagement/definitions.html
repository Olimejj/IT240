Internet:

A computer network spread golbaly following communication protocals basicaly it is a lot of servers and computers all talking together.

Web:

The World Wide Web is for sharing and displaying information. It is what allows us to browse content hosted all over the world. Its basically a network of documents that point to eachother.

URL:

A web pages address

Domain Name:

The domain name is the part of the URL which points to a specific website or domain of websites. Example lds.org

HTTP:

Hypertext transfer protocal is the protocal for transfering data on the web. HTTPS means the data is encrypted

IP Address:

Your IP address is your computers unique identity allowing you to send data to the IP address of another computer and recive a responce back to your IP address

DNS:

The domain name server keeps track of domains and the IP addresses that they represent.

W3C Recommendations:

W3C Recommendations are web standards layed out by the World Wide Web Consortium

XML, XHTML, HTML, HTML5:

Each of these are format standards for displaying information within the browser. HTML5 is the most comon one used for webpages.
