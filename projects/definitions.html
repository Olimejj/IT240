Internet:

The connection of lots of networks streaching accrose the world is what we call the internet. following communication protocals basicaly it is a lot of servers and computers all talking together.In the last decade this has expanded greatly to include devices like smart tv's, security cameras, smart phones, smart cars and so many other devices. This is what we call the internet of things.

Web:

The World Wide Web is for sharing and displaying information. It is what allows us to browse content hosted all over the world. Its basically a network of documents that point to eachother.

How are the Internet and the Web different?:

The Internet is all connections but the web is just those serving web pages. For example a fax machine can be connected to a network and accessable over the internet however it may not have a web page or browser interface.

ASCII Encoding:

ASCII is a table assigning values to symbols. It includes 128 charactors.This is how binary values are converted into text that we can read.

URL:

A web pages address in a readable rememberable form. For example the URL housin.byuh.edu is a lot easier to remember the IP address of 74.214.64.115

Domain Name:

The domain name is the part of the URL which points to a specific website or domain of websites. Example lds.org.

HTTP:

Hypertext transfer protocal is the protocal for transfering data on the web. HTTPS means the data is encrypted.

HTTPS:

Fuctions very similarly to HTTP however it adds a security by encrypting the data and authenticating via an ssl port.

IP Address:

Your IP address is your computers unique identity allowing you to send data to the IP address of another computer and recive a responce back to your IP address.

DNS:

The domain name server keeps track of domains and the IP addresses that they represent.

W3C Recommendations:

W3C Recommendations are web standards layed out by the World Wide Web Consortium.

XML, XHTML, HTML, HTML5:

Each of these are format standards for displaying information within the browser. HTML5 is the most comon one used for webpages today.
